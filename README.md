# YTIMP4

download MP3/MP4, videos, and live streams from YouTube

---

## INSTALL

git clone https://github.com/nixisaka/YTIMP4.git
cd YTIMP4
pip install -r requirements.txt

---

## RUN

Windows: python bootstrap.py
Linux/macOS: python3 ytimp4.py

---

## FEATURES

MP4 downloads (1080p+) · MP3 extraction with thumbnails · SRT subtitles · channel lookup & archive · download queue · auto subtitles · cookie scanning · download history · dark/light themes · batch downloads

---

## DEPENDENCIES 

Flask · yt-dlp · requests · beautifulsoup4 · aiohttp · colorama · google-api-python-client
